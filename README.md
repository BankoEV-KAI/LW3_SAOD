# LW3_SAOD - кольцевая очередь (динамическая релизация)

Задание на ЛР

Реализация структуры кольцевая очередь на основе динамического распределения памяти.

Функции, которые необходимо реализовать:

1. Стандартная работа 
	- проверка пустоты очереди;
	- проверка заполненности очереди;
	- добавление элемента в конец очереди;
	- удаление элемента из начала очереди;
	- вывод текущего состояния очереди на экран.
2. Автоматическое заполнение:
	- Пусть за единицу времени (например – 1 минуту) в очередь либо добавляется, 
либо удаляется случайное число элементов в количестве от 1 до 3-х. Одновременно добавление и удаление НЕ происходит.
